# TransferDAW – TFG
TransferDAW es una aplicación web orientada a la gestión de datos relaciones con el mundo del futbol. 
Dando como resultado una plataforma moderna, funcional y con potencial real de aplicación en el ámbito deportivo digital.

## Funcionalidades
  - Grstión de CRUD(insertar. actualizar, eliminar, visualizar) de datos deportivos.
  - Control de acceso con roles diferenciados.
  - Funcionalidades de auntetificación.
  - Visualización de estadísticas.
  - Descargar estadísticas de un jugador concreto.
  - Interfaz amigable, segura y sencilla.

## Stack Tecnológico
1- Máquina virtual Linux como entorno de tarbajo. 
2- Visual Studio Code como IDE
3- BBDD:
  - ERD plus para diseños
  - MariaDB para sistema gestor de BBDD 
4- Logica de negocio:
  - JAVA
  - Javadoc y doxygen para comentarios
  - Maven
  - Servlet 
5- Frontend:
  - JSP
  - CSS
  - JS
6- Tomcat para desplegar la web 
7- Crontab para copias de seguridad
